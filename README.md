# Omni
Development workflow orchestration tool


## Features
- Task runner
- Code scaffolding
- Environmental variable management
- Plugins

## Installation

### Cargo
```bash
cargo install omni
```

### npm
```npm
npm install -g omni
```

### Yarn
```yarn
yarn global add omni
```

### Bun
```bash
bun add -g omni
```

### Homebrew
```bash
brew install omni
```
